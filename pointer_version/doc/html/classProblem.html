Detailed Description

Problem with a unique name (id) and some statistics.

Invariant
  1. totalSubmissions = t >= e
  2. acceptedSubmissions = e >= 0
  3. ratio = (t+1)/(e+1)

Checks wether the implicit value (this) is to be ordered before the parameter (problem)

Precondition
true
Postcondition
true is returned if one holds:
  1. this->stats.ratio < problem->stats.ratio
  2. this->stats.ratio == problem->stats.ratio and this->id < problem->id
false is returned otherwise

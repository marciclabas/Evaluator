void User::parseSubmission ( prb::ID  problemID,

Updates the user's stats according to the given result of a submission.

Precondition
problemID is a valid problem whose prerequisites are held by the user
Postcondition
The problem sumission count is incremented by 1. If result equals result::accepted, a) The problem is removed from solvable and added to solved, and b) If the user finished the course, said course is notified

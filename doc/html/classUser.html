void User::parseSubmission ( prb::ID  problemID,
prb::result  r 
)

Updates the user's stats according to the given result of a submission.

Precondition
problemID is a valid problem whose prerequisites are held by the user
Postcondition
  1. The problem sumission count is incremented by 1
  2. If result equals result::accepted,
    • The problem is removed from solvable and added to solved
    • If the user finished the course, said course is notified
